# S-033: Panel de Super Administrador

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Media
**Estimación**: 3 días
**Sprint**: Época 5
**Estado**: Pendiente

## Historia de Usuario
**Como** administrador de la plataforma FleetFlow
**Quiero** gestionar todos los inquilinos desde un panel central
**Para que** pueda monitorear la plataforma y dar soporte a clientes

## Criterios de Aceptación
- [ ] Ver todos los inquilinos con métricas clave
- [ ] Buscar y filtrar inquilinos
- [ ] Ver detalles y uso del inquilino
- [ ] Suplantar inquilino para soporte
- [ ] Ajustar manualmente planes/límites de inquilinos
- [ ] Ver análisis de toda la plataforma
- [ ] Gestionar problemas de suscripción
- [ ] Enviar anuncios a todos los inquilinos

## Secciones del Panel

### Gestión de Inquilinos
- Lista de inquilinos con estado, plan, MRR
- Vista de detalle de inquilino
- Suplantación (iniciar sesión como inquilino)
- Anulaciones manuales de plan

### Análisis de Plataforma
- MRR total, ARR
- Conteo de clientes por plan
- Tasa de abandono
- Nuevos registros
- Conversiones de prueba

### Herramientas de Soporte
- Problemas de suscripción
- Pagos fallidos
- Tickets de soporte (futuro)
- Logs de actividad

## Definición de Terminado
- [ ] Rol de super admin y autenticación implementados
- [ ] Interfaz de gestión de inquilinos funcionando
- [ ] Panel de análisis de plataforma implementado
- [ ] Suplantación con registro de auditoría funcionando
- [ ] Pruebas cubren funciones de admin (>95% cobertura)
- [ ] Documentación de operaciones de admin
