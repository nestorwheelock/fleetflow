# S-009: Reservas en Línea

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Alta
**Estimación**: 3 días
**Sprint**: Época 2
**Estado**: Pendiente

## Historia de Usuario
**Como** cliente
**Quiero** reservar un alquiler en línea
**Para que** pueda reservar un vehículo cuando me convenga

## Criterios de Aceptación
- [ ] Puede seleccionar fechas de recogida y devolución
- [ ] Puede elegir vehículo disponible
- [ ] Puede agregar extras opcionales (GPS, silla de bebé)
- [ ] Puede ver desglose de precio antes de reservar
- [ ] Puede guardar reservación como borrador
- [ ] Recibe email de confirmación
- [ ] Puede cancelar/modificar reservación (con política)

## Definición de Terminado
- [ ] Flujo de reserva del cliente implementado
- [ ] Integración de selector de fecha/hora funcionando
- [ ] Sistema de selección de extras implementado
- [ ] Calculadora de precios en tiempo real funcionando
- [ ] Emails de confirmación implementados
- [ ] Lógica de cancelación/modificación funcionando
- [ ] Pruebas escritas y pasando (>95% cobertura)
- [ ] Documentación actualizada
