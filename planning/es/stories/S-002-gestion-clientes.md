# S-002: Gestión de Clientes

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Alta
**Estimación**: 2 días
**Sprint**: Época 1
**Estado**: Pendiente

## Historia de Usuario
**Como** agente de alquiler
**Quiero** gestionar información de clientes
**Para que** pueda rastrear arrendatarios, sus documentos e historial de alquileres

## Criterios de Aceptación
- [ ] Puede agregar clientes con información de contacto y dirección
- [ ] Puede registrar detalles de licencia de conducir (número, estado, expiración)
- [ ] Puede subir fotos de licencia y otros documentos
- [ ] Puede ver historial de alquileres del cliente
- [ ] Puede buscar clientes por nombre, email, teléfono o licencia
- [ ] Puede marcar clientes (VIP, prohibido, etc.)

## Definición de Terminado
- [ ] Modelo de Cliente creado con todos los campos
- [ ] CRUD completo de clientes implementado
- [ ] Sistema de carga de documentos funcionando
- [ ] Pruebas escritas y pasando (>95% cobertura)
- [ ] Documentación actualizada
