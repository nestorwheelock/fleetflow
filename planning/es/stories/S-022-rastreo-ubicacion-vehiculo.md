# S-022: Rastreo de Ubicación de Vehículos en Tiempo Real

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Alta
**Estimación**: 3 días
**Sprint**: Época 4
**Estado**: Pendiente

## Historia de Usuario
**Como** cliente
**Quiero** ver dónde está mi vehículo alquilado en un mapa
**Para que** pueda encontrarlo en un estacionamiento o si es robado

## Criterios de Aceptación
- [ ] Puede ver ubicación del vehículo en mapa en tiempo real
- [ ] Puede ver última ubicación conocida si está sin conexión
- [ ] Puede obtener direcciones a la ubicación del vehículo
- [ ] El mapa se actualiza automáticamente cada 30 segundos
- [ ] Puede ver dirección/rumbo del vehículo
- [ ] Historial de ubicación para el período de alquiler actual

## Definición de Terminado
- [ ] Pantalla de mapa con MapBox/Google Maps implementada
- [ ] Ubicación en tiempo real vía WebSocket funcionando
- [ ] Marcador de vehículo con indicador de dirección implementado
- [ ] Integración de navegación/direcciones funcionando
- [ ] Línea de tiempo de historial de ubicación implementada
- [ ] Caché de ubicación sin conexión funcionando
- [ ] Pruebas escritas y pasando (>95% cobertura)
- [ ] Documentación actualizada
