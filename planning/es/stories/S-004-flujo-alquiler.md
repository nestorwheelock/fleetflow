# S-004: Flujo de Alquiler (Check-out/Check-in)

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Alta
**Estimación**: 3 días
**Sprint**: Época 1
**Estado**: Pendiente

## Historia de Usuario
**Como** agente de alquiler
**Quiero** procesar recogidas y devoluciones de vehículos
**Para que** pueda rastrear el ciclo completo del alquiler

## Criterios de Aceptación
- [ ] Puede hacer check-out de vehículo con lectura de odómetro
- [ ] Puede registrar condición del vehículo antes del alquiler
- [ ] Puede hacer check-in de vehículo con odómetro de devolución
- [ ] Puede registrar condición posterior al alquiler y daños
- [ ] Puede calcular cargos finales (millas extra, combustible, daños)
- [ ] Puede generar recibo

## Definición de Terminado
- [ ] Flujo de trabajo de check-out implementado
- [ ] Flujo de trabajo de check-in implementado
- [ ] Modelo de reporte de condición creado
- [ ] Lógica de cálculo de cargos funcionando
- [ ] Pruebas escritas y pasando (>95% cobertura)
- [ ] Documentación actualizada
