# S-019: Autenticación Móvil del Cliente

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Alta
**Estimación**: 2 días
**Sprint**: Época 4
**Estado**: Pendiente

## Historia de Usuario
**Como** cliente
**Quiero** iniciar sesión en mi cuenta desde mi teléfono
**Para que** pueda gestionar alquileres en movimiento

## Criterios de Aceptación
- [ ] Puede registrar nueva cuenta en móvil
- [ ] Puede iniciar sesión con email/contraseña
- [ ] Puede usar inicio de sesión biométrico (Face ID/huella)
- [ ] Puede restablecer contraseña desde móvil
- [ ] La sesión persiste de forma segura en el dispositivo
- [ ] Puede cerrar sesión y cambiar cuentas

## Definición de Terminado
- [ ] Proyecto React Native configurado con Expo
- [ ] Pantallas de autenticación implementadas
- [ ] Integración de autenticación biométrica funcionando
- [ ] Almacenamiento seguro de tokens implementado
- [ ] Cliente API para backend Django configurado
- [ ] Pruebas escritas y pasando (>95% cobertura)
- [ ] Documentación actualizada
