# S-003: Sistema de Calendario de Reservaciones

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Alta
**Estimación**: 3 días
**Sprint**: Época 1
**Estado**: Pendiente

## Historia de Usuario
**Como** agente de alquiler
**Quiero** ver y gestionar reservaciones en un calendario
**Para que** pueda ver disponibilidad y prevenir dobles reservaciones

## Criterios de Aceptación
- [ ] Puede ver calendario mensual/semanal/diario de reservaciones
- [ ] Puede ver qué vehículos están reservados en qué fechas
- [ ] Puede hacer clic para crear nuevas reservaciones
- [ ] Puede arrastrar para reprogramar reservaciones
- [ ] No puede crear reservaciones superpuestas para el mismo vehículo
- [ ] Puede filtrar calendario por categoría de vehículo
- [ ] Puede ver horarios de recogida y devolución
