# S-011: Firma Electrónica de Contratos

**Tipo de Historia**: Historia de Usuario
**Prioridad**: Alta
**Estimación**: 2 días
**Sprint**: Época 2
**Estado**: Pendiente

## Historia de Usuario
**Como** cliente
**Quiero** firmar el contrato de alquiler electrónicamente
**Para que** pueda completar el papeleo antes de la recogida

## Criterios de Aceptación
- [ ] Puede ver términos del contrato en línea
- [ ] Puede proporcionar firma electrónica (dibujada o escrita)
- [ ] Marca de tiempo de firma e IP registradas
- [ ] Contrato firmado almacenado como PDF
- [ ] Contrato enviado por email al cliente
- [ ] El personal puede verificar estado de firma

## Definición de Terminado
- [ ] UI de captura de firma electrónica implementada
- [ ] Sistema de verificación y almacenamiento de firma funcionando
- [ ] Generación de PDF con firma implementada
- [ ] Entrega de contrato por email funcionando
- [ ] Pruebas escritas y pasando (>95% cobertura)
- [ ] Documentación actualizada
