# S-033: Super Admin Dashboard

**Story Type**: User Story
**Priority**: Medium
**Estimate**: 3 days
**Sprint**: Epoch 5
**Status**: Pending

## User Story
**As a** FleetFlow platform administrator
**I want to** manage all tenants from a central dashboard
**So that** I can monitor the platform and support customers

## Acceptance Criteria
- [ ] View all tenants with key metrics
- [ ] Search and filter tenants
- [ ] View tenant details and usage
- [ ] Impersonate tenant for support
- [ ] Manually adjust tenant plans/limits
- [ ] View platform-wide analytics
- [ ] Manage subscription issues
- [ ] Send announcements to all tenants

## Dashboard Sections

### Tenant Management
- Tenant list with status, plan, MRR
- Tenant detail view
- Impersonation (login as tenant)
- Manual plan overrides

### Platform Analytics
- Total MRR, ARR
- Customer count by plan
- Churn rate
- New signups
- Trial conversions

### Support Tools
- Subscription issues
- Failed payments
- Support tickets (future)
- Activity logs

## Definition of Done
- [ ] Super admin role and authentication
- [ ] Tenant management interface
- [ ] Platform analytics dashboard
- [ ] Impersonation with audit logging
- [ ] Tests cover admin functions (>95% coverage)
- [ ] Documentation for admin operations
